# Team Checklist

## Before Work
- [ ] `git checkout <branch>`
- [ ] `git pull` if needed
- [ ] Confirm task scope
- [ ] Confirm no file overlap

## During Work
- [ ] Follow assigned module only
- [ ] Keep changes small
- [ ] Test locally if possible
- [ ] Commit each milestone

## Before Handover
- [ ] `git status` clean or expected
- [ ] Commit message clear
- [ ] Send summary of changed files
- [ ] Note any blockers
