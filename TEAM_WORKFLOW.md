# Team Workflow

## Branches
- `codex/ethunder243-droid` ? auth + role routing
- `codex/maulana-bintang` ? schema + RLS + storage + realtime

## Workflow
1. `git checkout <branch>`
2. Kerja sesuai task branch
3. `git add .`
4. `git commit -m "..."`
5. Kirim hasil untuk review
6. Merge ke `master` bila aman

## Rules
- 1 branch = 1 scope
- Jangan edit file yang sama bersamaan
- Commit tiap milestone
